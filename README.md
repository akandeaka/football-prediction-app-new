# Football Prediction Engine

Rule-based prediction engine for football matches.

## How to Run
1. Open this repository in Codespaces.
2. Open Terminal.
3. Run: `python prediction_engine.py`

## Rules
1. Home Win (1.20-1.29) vs Away (10.00+)
2. Home Win (1.30-1.35) vs Away (12.00+)
3. Home/Draw (1.90-1.99)
4. Over 1.5 Goals (Specific Draw/Away conditions)
5. 1st Half Under 1.5 (H2H Stats)
